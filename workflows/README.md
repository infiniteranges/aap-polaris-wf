# AAP Workflow Templates

This directory contains documentation for AAP workflow job templates.

Each workflow type will have:
- Basic workflow (auto-approve)
- Workflow with approval node

## Planned Workflows

- `terraform-basic.md` - Terraform execution without approval
- `terraform-with-approval.md` - Terraform execution with approval
- `terragrunt-basic.md` - Terragrunt execution without approval
- `terragrunt-with-approval.md` - Terragrunt execution with approval
- `tfc-basic.md` - Terraform Cloud execution without approval
- `tfc-with-approval.md` - Terraform Cloud execution with approval
